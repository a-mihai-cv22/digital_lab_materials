# digital_lab_materials

English:

This Github repository functions as the complementary material to the Integrated Digital Circuits laboratory guide written by Antonescu Mihai and Popescu George-Vladut at the National University for Science and Technology Politechnica Bucharest.
It contains all the example projects and different testbenches used to help student understanding


Romanian:

Aceasta arhiva Github reprezinta materialele suplimentare pentru ghidul de laborator de Circuite Integrate Digitale, scris de Antonescu Mihai si Popescu George-Vladut de la Universitatea Nationala de Stiinta si Tehnologie Politehnica Bucuresti.
Acesta contine toate problemele exemplu si diferite module de testare, avand ca scop imbunatatirea procesului de invatare.
